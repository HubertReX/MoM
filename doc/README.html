Misadventures of Malachi

In the Kingdom of Knaveshire, a land teeming with monsters, magic, and questionable hygiene practices, our not-so-heroic hero, Malachi, finds himself burdened with a curse that makes him irresistibly attractive to danger and utterly incompetent at solving any problem. Armed with an ironic sense of humor and an uncanny ability to stumble into trouble, Malachi embarks on an epic quest to rid himself of the curse and maybe, just maybe, save the kingdom in the process.

From author

This is an early-stage of RPG game I started creating in early 2023. The first version of this game was built for the terminal and was called SSIS. As my ideas for the game grew, I decided to switch to a graphical interface and make it playable in the browser 

For now, this game is more of a technical demo - there is no plot or a clear goal, yet. I'm going eventuall add a reach story (the draft version is ready). But first, I need to add more fundamental features like: trade, skill progression, game save/load, music and sfx, and full dialog system.
